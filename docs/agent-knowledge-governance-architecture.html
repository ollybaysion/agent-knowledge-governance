agent-knowledge-governance — 지식이 쌓이는 구조

사내 도메인 지식의 SSoT. 지식은 세 경로로 들어와(신규 저작 · 기존 문서 수집 · 스키마·팩트), 관문을 통과해 git store에 진실원으로 앉고, 슬롯 티어 생애를 거쳐 로컬로 주입된다.

스키마 검증

타입 스키마로 검증(수제 ajv). 전 층위 additionalProperties:false — 모르는 키·버전 거부.

격리(quarantine)

수집물은 예외 없이 quarantined 착지. 안 보임이 아니라 안 나감 — rendered/·번들에서 제외, 주입·배포 차단.

검토 대기열

사람이 슬롯/배치 단위로 열람·채택. proposals·intake 큐 모두 git 안이라 재시작에도 살아남고 diff로 읽힘.

채택 · 활성화

채택=inferred 반영(에이전트 경로 한계). 수집물 activate=키워드 확정+렌더(approver).

▼ 진실원 저장 + 파생물 렌더가 한 커밋
③SSoTgit 레포 store/ — 운영 DB 없음

진실원 문서 = store/<type>/<id>.json

봉투 (5키 고정)
schema · id · keywords · status · body — 라우팅·주입 메타
└ body — 타입별 알맹이
팩트 (기계)원천과 대조로 참/거짓. 티어 없음·신선도만. catalog-push 자동.
슬롯 (사람)해석 = 티어드 값 {text,tier,evidence,by,at}. 검토·승격 단위.

파생물 (같은 커밋에 자동 생성)

render(json)의 결정적 산출 — 어긋난 상태가 존재 불가.

rendered/<type>/index.json 프로바이더 계약

rendered/<type>/docs/<id>.md 주입용 md

감사 = git 이력 그 자체(author·시각·경로·메시지). GET /api/audit=git log.

유일한 비-git 상태

users.json (0600) — 토큰 sha256 해시 + 역할. 시크릿이라 git 밖.

④슬롯 티어 생애사람 지식의 신뢰도 (이 전이가 곧 쓰기 검증 사양)
scaffold

→
inferred
채택·편집. evidence 1개 이상 필수. 에이전트는 여기까지.
→
confirmed
promote — approver 전용. text·evidence 불변.

신뢰 철회 — 원천 소멸(자동) or 수동 폐기. 내용 보존, 렌더·주입 제외.

못 박힌 규칙: confirmed로 들어오는 화살표는 promote 하나뿐 — 어떤 쓰기 경로도 confirmed를 직접 생성 못 함. confirmed 슬롯을 고치면 자동 강등(→inferred). agent 토큰은 티어드 값을 직접 못 씀(제안만). deprecated 복원 = 재채택 → 항상 inferred(재검토 강제), 제거 확정 = approver.

▼ sync(미러 당김) · 키워드 매치 시 주입
⑤소비각자 로컬(CC)로 주입

akg CLI

sync 미러 당김 · propose 제안 푸시 · catalog-push 팩트 갱신

미러 = 로컬 캐시 ~/.claude/akg/<type>/ — 서버가 죽어도 마지막 미러로 계속 동작.

키워드 매치 → 주입

키워드 매치는 항상 로컬 미러 인덱스. 본문은 두 모드:

A모드 — 미러 md 주입(claude-hooks 무수정, 기본)

B모드 — 매치 시 허브 신선본 fetch(하드 타임아웃 → 실패 시 미러 폴백)

한눈 요약 — 이 구조를 지탱하는 불변식 4

서버가 유일한 writer

모든 변이는 서버 API → 쓰기 큐 직렬화 → git commit 1건. 단일 프로세스라 머지 충돌이 원천 부재.

모르는 키 거부

JSON 진실원 + 전 층위 스키마 검증. md 시절 "무경고 유실"을 스키마 마이그레이션으로 드러낸다.

사람만 승격

agent=제안만, editor=채택·편집(inferred), approver=승격(confirmed)·삭제 확정. 신뢰는 사람 관문을 통과해야 오른다.

격리는 "안 나감"

수집물은 quarantined 착지 → rendered/·번들에서 제외. 검토 전 외부 텍스트가 주입·배포 표면에 못 오른다(프롬프트 주입 방어).
