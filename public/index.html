지식 레지스트리

Claude Code 같은 AI 에이전트가 사내 도메인 지식을 프롬프트에 자동으로 주입받도록 모아 두는 저장소입니다. 검증 없이 주입하면 틀린 지식이 모든 세션으로 퍼지므로, 지식 한 조각마다 신뢰 등급(티어)을 매기고 관리자가 검증해 올리는 관문을 둡니다. 오른쪽 다이어그램이 그 등급의 일생입니다.

관리 단위

문서 = 대상 하나(테이블·커맨드·용어·스킬). 안은 두 종류입니다:

  • 팩트 — 기계 수집(컬럼·타입). 원천과 대조로 검증 → 티어 없음.
  • 슬롯 — 관리자의 해석(용도·설명). 검토로 검증 → 티어는 여기만.

검토·승격은 슬롯 단위 (스킬만 문서 단위).

문서 종류
db-schema테이블 · 컬럼
msg-format설비 커맨드
domain-doc도메인 용어
domain-skill조회 절차
활용
  1. 에이전트가 코드베이스를 읽고 초안을 제안
  2. 관리자가 추정으로 반영하고, 원천과 대조해 확정
  3. 확정된 지식이 모든 CC 세션 프롬프트에 주입

왼쪽 트리로 문서 탐색 · 개요로 검토 현황 · 검토 대기열에서 승격.
